# Tasks_automation
Directory name changer - changes names with the given structure:

1. Main_dir
     - Dir
          - Rev_A
     - Dir2
          - Rev_A
          
to:

2. Main_dir
     - Dir
          - Rev_B
     - Dir2
          - Rev_B

User give the program the old name (Rev_A), the new name (Rev_B) and the directory path(to Main_dir).
